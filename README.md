# VK-бот для рекомендаций фильмов

Чат-бот для платформы VK, который подбирает персональные рекомендации фильмов и сериалов на основе оценок пользователя («Нравится» / «Не нравится»). Для холодного старта (минимальная история оценок) используется гибридная модель **Score-Level Fusion** (EASE + ItemKNN).

## Возможности
- Регистрация и хранение профиля пользователя (SQLite)
- Подбор фильмов по жанру с интерактивной оценкой
- Персональные рекомендации на основе модели Fusion
- Просмотр популярных фильмов
- Связь с поддержкой
- Удаление учётной записи

## Модели
Файлы предобученных моделей (`ease.pkl`, `knn.pkl`, `fusion.pkl`) весят более 1 ГБ суммарно и не хранятся в репозитории из-за ограничений GitHub. Все `.pkl`-файлы моделей загружены на Google Диск:

Ссылка на модели: https://drive.google.com/drive/folders/1g84DX85gf0YjZvXaCONWpn1yZistGkRV?usp=sharing

После скачивания поместите файлы в папку `models/` в корне проекта перед запуском бота.

## Запуск
1. Установить зависимости: `pip install -r requirements.txt`
2. Создать `config.py` с переменной `TOKEN` (VK-токен)
3. Скачать модели по ссылке выше и положить в `models/`
4. Запустить: `python bot.py`
